# 10daysthai — Read Thai Redesign

Книжный редизайн интерактивного учебника тайского алфавита (на 10 дней).

## Файлы

| Файл | Что это |
|---|---|
| `readThai.html` | Точка входа. Подключает React + Babel и собирает приложение из JSX. |
| `readThai-standalone.html` | **Один файл со всем внутри** (CSS, JS, шрифты инлайн). Открывается двойным кликом. Используйте для демо/деплоя. |
| `design-system.html` | Спека: цвет, типографика, сетка, шаг, компоненты, движение, тон. |
| `data.jsx` | Контент уроков, согласные/гласные, упражнения. |
| `components.jsx` | Все React-компоненты (карточки, плеер, флеш-карты, тренажёр голоса). |
| `app.jsx` | Корневой компонент, роутинг (home / lesson / appendix), сайдбар. |
| `tweaks-panel.jsx` | Панель Tweaks (палитра, размер тайских глифов, стиль упражнений). |

## Запуск

Откройте `readThai.html` в браузере — этого достаточно. Для офлайн-демо используйте `readThai-standalone.html`.

## Архитектура

- **Без сборки.** React 18 + Babel Standalone через CDN. JSX исполняется в браузере.
- **Стили** — один большой `<style>`-блок в `readThai.html`. Переменные CSS — в `:root` и `html[data-theme="dark"]`.
- **Шрифты**: `Cormorant Garamond` (regular/medium + italic) — всё тело. `Source Serif 4 italic` — точечно для акцентных слов в крупных заголовках. `IBM Plex Sans Thai Looped` (400/500) — тайские глифы.

## Дизайн-система

См. `design-system.html`. Кратко:

- **Палитра**: шафран на тёплой бумаге. Альтернативы (Индиго / Нефрит / Тушь) — в Tweaks.
- **Типографика**: одна гарнитура для всего, без жирных весов. Курсив — только для смысловых акцентов и чисел в стиле книг.
- **Сетка**: 280px sidebar + центральная колонка max 1080 / статья 720.
- **Движение**: сдержанное; 120ms hover, 200ms lift, 500ms flip. Никаких parallax.

## Тон

Наставник, не тренер. Без эмодзи, без восклицаний. Числа прописью курсивом: *день первый*.
